How to include a random number tag into a label design

Generate random numbers using the [#R] automation tag.

Customize random number generation minimum and maximum values by going to
Toolbar > Automation Tags

When you include automation tags in your label content, you can render tags to permanent values by going to Toolbar > Automation Tags > Render Tags Now
